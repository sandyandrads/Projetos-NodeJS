O que você precisar para iniciar seu projeto

1 - criar a pasta
2 - dar npm init na pasta
4 - criar o arquivo js
3 - instalar o express
4 - instalar nodemon para melhorar produtividade assim não precisar reiniciar servidor
5 - você se perdeu quando criou routes, não esquecer as mudanças de end-points após criar routes novamente
6- metodo writeFile substitui o objeto JSON, metodo appendFile adiciona mais um objeto
